## Quick Start
Follow these steps to run the app:

1. Clone the repository:
   ```bash
   git clone https://github.com/wwwdmtr/react-native-expo-simple_cards-app.git

2. Install dependencies
   ```bash
   yarn install
   
3. Start the app
   ```bash
   npx expo start
